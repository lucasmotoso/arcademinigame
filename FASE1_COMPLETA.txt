🚀 AGORA VOCÊ PODE:
═══════════════════════════════════════════════════════════════════════════

1. INICIAR DEVELOPMENT SERVER:
   $ npm run dev
   
   Isso iniciará o esbuild em watch mode, reconstruindo automaticamente
   toda vez que você salvar um arquivo. O JavaScript será bundled e
   minificado em tempo real.

2. BUILD PARA PRODUÇÃO:
   $ npm run build
   
   Cria uma versão otimizada em dist/main.js com minificação completa
   e source maps para debugging.

3. VERIFICAR CÓDIGO:
   $ npm run lint
   
   Verifica erros e avisos no código JavaScript usando ESLint.

4. FORMATAR CÓDIGO:
   $ npm run format
   
   Formata automaticamente todos os arquivos JS usando Prettier.

5. PREVIEW DO SITE:
   $ npm run preview
   
   Inicia um servidor HTTP na porta 8080 para visualizar o site.
