Esta é uma versão JavaScript do Pong.

Aperte 1 para single player.
Aperte 2 para double player.
Aperte 0 ver o computador jogar sozinho.

Jogador 1 se movimenta usando Q e A keys.
Jogador 2 se movimenta usando P e L keys.

Esc para abandonar o jogo.
