🔧 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════════

"npm: command not found"
→ Instale Node.js 18+: https://nodejs.org/

".\init.ps1 não é reconhecido"
→ Execute: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
→ Ou use: python init-structure.py

"Port 8080 em uso"
→ Execute com outra porta: npm run dev -- --port 3000

"Erro no npm install"
→ Tente: npm cache clean --force && npm install
